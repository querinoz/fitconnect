## The elevator pitch

Future ($199/mo) gives you one generalist strength coach. Trainerize gives coaches the tools but no athletes. Strava gives you a feed but no coaching. **FitConnect is the only platform where 12,418 verified specialists across 10 sports — yoga, surf, BJJ, climbing, cycling and more — meet the science-grade dashboard (HRV, sleep correlation, recovery-aware booking, AI workout adjustments) that elite athletes pay university programs for.** Free to start, free to talk to every coach, €12/mo when you&rsquo;re ready.

## ★ Why FitConnect

- **The Specialist Standard™.** Every coach interviewed by a senior FitConnect coach. Every certificate verified against the issuing body. 62% of applicants rejected. The badge means something.
- **Recovery-aware booking.** HRV + sleep from Apple Watch / Garmin / Whoop flow straight to your coach&rsquo;s plan — bad night, lighter session, no debate.
- **Multi-sport athlete identity.** Train Vinyasa Monday, BJJ Wednesday, intervals Saturday — one identity, one unified recovery score, one progress graph.
- **Free 15-minute intro with every coach.** Never pay before you&rsquo;ve looked your human in the eye. Switch any time, your athlete profile travels.
- **Coaches keep 85%.** The highest take-home on any marketplace. We post our P&amp;L.
- **Shipping weekly.** The product you sign up for today looks better in two weeks. Honest roadmap below.

## 🧱 Tech stack

| Layer | Choice | Why |
| --- | --- | --- |
| Framework | **Next.js 14** (App Router) | RSC + static prerender for marketing pages |
| Language | **TypeScript 5** | Strict, end-to-end |
| Styling | **Tailwind CSS 3** | Design tokens in `tailwind.config.ts`, custom keyframes |
| Animation | **Framer Motion 11** | Hero parallax, scroll-reveal, micro-interactions |
| Charts | **Recharts 2.13** | Area/Bar/Line for dashboard analytics |
| Icons | **lucide-react** | Consistent line iconography, the Dumbbell is the logo mark |
| Primitives | **Radix UI** | Dialogs, dropdowns, tabs, avatars |
| Type fonts | **Inter** &amp; **Space Grotesk** | Body + display, paired with `font-feature-settings: ss01, cv01` |
| State | **Zustand 5** | Tiny, ergonomic, no providers |

## 🚀 Quick start

```bash
git clone https://github.com/querinoz/fitconnect.git
cd fitconnect
npm install
npm run dev            # → http://localhost:3001
```

No environment variables required for the demo — content is seeded in `lib/data.ts`.

| Script | Action |
| --- | --- |
| `npm run dev` | Dev server on port **3001** |
| `npm run build` | Production build (19 static pages) |
| `npm run start` | Production server on port **3001** |
| `npm run lint` | Next.js ESLint |

## 📦 Deployment

```bash
# Vercel — zero-config, recommended
vercel
```

```bash
# Netlify — using @netlify/plugin-nextjs
netlify deploy --build
```

```bash
# Docker — multi-stage Node 20 alpine, EXPOSE 3001
docker build -t fitconnect .
docker run -p 3001:3001 fitconnect
```

Configs are already committed: `vercel.json`, `netlify.toml`, `Dockerfile`.
